# Observation Visualization

## Goal
Show users how their Observation metrics change over time through charts, so trends become visible
without manually reviewing raw Records. Grouped as one epic because the features only deliver on that
goal together: a shared rendering foundation, one chart type per Metric type (Numeric, Boolean, Enum,
Text), time-range selection, tap-to-detail, and per-metric visibility control all build on each other,
and none is independently useful as a standalone capability.
